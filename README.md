## GAME CONTROLS
~~~~~~~~~~~~~~~

Keyboard Player controls:
- UP --> W key
- LEFT --> A key
- RIGHT --> D key
- DOWN --> S key
- GUN SHOT --> SPACE key
- CANNON ROTATION RIGHT --> ⮕ key
- CANNON ROTATION LEFT --> ⬅ key
